# 🕸️ Largest U.S. Companies Web Scraping Project

## 📌 Project Overview

This project demonstrates web scraping using Python to extract data from Wikipedia's list of the largest companies in the United States by revenue.

Using Requests and BeautifulSoup, the data was collected from a web page, converted into a structured format with Pandas, and exported as a CSV file for further analysis.

---

## 🎯 Project Objective

The main objectives of this project were:

- Extract data from a live website
- Practice web scraping techniques using Python
- Parse HTML content with BeautifulSoup
- Convert extracted data into a structured dataset
- Export the final dataset to CSV format

---

## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook

---

## 📊 Data Source

Wikipedia: List of Largest Companies in the United States by Revenue

---

## 🔄 Project Workflow

1. Sent an HTTP request to the Wikipedia page.
2. Parsed the HTML content using BeautifulSoup.
3. Identified and extracted the required table.
4. Converted the extracted data into a Pandas DataFrame.
5. Cleaned and organized the dataset.
6. Exported the final data to a CSV file.
